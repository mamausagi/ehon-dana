# 同梱絵本（本棚に最初から並べる絵本）

このフォルダに `.ehon.json`（音声づくり版で「📤 絵本ファイル書き出し」したもの）を置き、
`index.json` にファイル名を並べると、**えほんだなを開いた全端末の本棚に自動で入る**。

## 手順

1. 音声づくり版（デスクトップの「えほんスタジオ（音声づくり）」）で絵本を開き、
   音声・表紙を仕上げて「📤 絵本ファイル書き出し」→ `.ehon.json` を保存
2. その `.ehon.json` をこのフォルダ（`personal/ehon-dana/books/`）に入れる
3. `index.json` にファイル名を追記（例）:
   ```json
   ["neko.ehon.json", "otsukai.ehon.json"]
   ```
4. デプロイ（`deploy-app` スキル / `git push`）

## 仕組み

- アプリ初回起動時に `books/index.json` を読み、各ファイルを本棚へ取り込む（`loadBundled()`）
- **book.id で重複判定**。同梱本を更新（同id・新しい `updatedAt`）すれば入れ替わる
- 端末で手動 📥 取り込みした絵本は同梱本と別管理（消えない）
- Claudeに「この絵本を えほんだな に追加して」と言えば、上記2〜4を代行できる
